Rotas para Itens (Items):

  • GET /items: Retorna todos os registros válidos de itens.
  • GET /items/<int:id>: Retorna um registro único de item identificado pelo id.
  • POST /items: Cadastra um novo registro de item.
  • DELETE /items/<int:id>: Marca como apagado um registro único de item identificado pelo id.
  • PUT/PATCH /items/<int:id>: Edita um registro de item identificado pelo id.

Rotas para Proprietários (Owners):

  • GET /owner: Retorna todos os registros válidos de proprietários.
  • GET /owner/<int:id>: Retorna um registro único de proprietário identificado pelo id.
  • POST /owner: Cadastra um novo registro de proprietário.
  • DELETE /owner/<int:id>: Marca como apagado um registro único de proprietário identificado pelo id.
  • PUT/PATCH /owner/<int:id>: Edita um registro de proprietário identificado pelo id.
  • GET /owner/<int:id>/items: Retorna todos os itens relacionados a um proprietário específico.

Outras Rotas:

  • GET /items/dados/<int:item_id>: Retorna um item com seus detalhes, incluindo detalhes do proprietário.
  • GET /item/search/<string:item_name>: Busca itens com base em um nome fornecido.
  • POST /contact: Cadastra um novo contato.
